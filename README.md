# Multi-Department-Business-Analytics-Dashboard
A complete business analytics dashboard project built in Excel using Power Query, Power Pivot, and Data Modeling. This project transforms raw business data into interactive dashboards that provide clear insights for decision-making.

It includes dashboards for Sales, Customers, Products, Supply Chain, Marketing, and HR. Key features include KPI cards, pivot tables, slicers, dynamic charts, and performance tracking metrics.

Used data cleaning, transformation, and relationship modeling to connect multiple tables and create accurate reports.

**Features:**
Data Cleaning & Transformation
Data Model Relationships
KPI Cards & Measures
Interactive Dashboards
Sales Analysis
Customer Insights
Product Performance
Supply Chain Tracking
Marketing ROI Analysis
HR Metrics Dashboard
Tools Used:
Microsoft Excel
Power Query
Power Pivot
Pivot Tables
Slicers & Charts

**Outcome:**

Turned raw business data into actionable insights through professional dashboards for smarter business decisions.
